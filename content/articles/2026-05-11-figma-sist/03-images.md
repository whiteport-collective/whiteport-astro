# Image prompts — Figma sist, inte först

## Illustration: Två vägar — en bild

En bild med två parallella flöden staplade ovanpå varandra.

**Övre rad — Figma first (röda pilar, negativa labels):**

Project plan (shallow context)
→🔴 Figma design (details before substance)
→🔴 Vibe coding (endless editing · spaghetti code)
→🔴 Full scale development (poor context · more spaghetti)
→🔴 Backfilling content (added complexity)
→ **Outcome: Messy delivery. Unsustainable future.**

**Nedre rad — Figma last / WDS (gröna pilar, positiva labels):**

Project brief + trigger mapping (solid clarity · target group match)
→✅ Wireframes + spec + content (rich context · challenges assumptions)
→✅ Agentic development (specs box in the agent · re-generate on change)
→✅ Visual adjustment in Figma (open code in Figma · IDs maintained)
→ **Outcome: Solid deliveries. Ready for iteration.**

---

**Visuell riktning:**
- Tydlig uppdelning: röd rad vs grön rad
- Figma-logotypen syns i båda raderna men med olika position — tidig (röd) vs sen (grön)
- Pilar i rött resp. grönt
- Svart/mörk bakgrund, clean typografi
- Läsbar som standalone utan artikeltext
