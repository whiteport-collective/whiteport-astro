I asked my agent to count.

The fix took less than an hour. Three prompts:

𝟭. "How many tokens did my MCP servers load at startup? Break it down per server and per tool."

𝟮. "Build a script that lets me use my MCP when I want, without loading it at startup."

𝟯. "Extract each tool call into a separate skill file, then link to it from the main skill where it's used."

That's it.

You find the link to the instructions in the first comment, feel free to share how many tokens you could save in every conversation!
