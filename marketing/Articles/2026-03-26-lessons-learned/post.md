## LinkedIn (Mårten personlig)

After 8 months of building with AI, I've learned that clarity is the most expensive currency in the AI era.

Not the model. Not the tools. Clarity.

Here are 8 lessons from running an AI agent framework in a real design team. The last one will probably surprise you.

1. Strategy is no longer optional for designers. When producing things is effortless, deciding what to produce becomes the real skill.

2. Map user psychology before you design. Trigger mapping changed everything. Every feature now traces back to a real human driving force — not a hunch.

3. Figma is not a design tool anymore. It's an agent training tool. You open it when the agent gets it wrong — not to start from scratch.

4. Specifications are the new code. The spec isn't a handoff document. It IS the product. Code is just a rendering of it.

## Instagram (@whiteport + Mårten som collaborator)

Most AI design tools make individuals faster.
They don't make teams better.

After months running an AI agent framework in a real design team — the biggest surprise wasn't the tech.

It was the order.

Clarity → Psychology → Scenarios → Specs → Code.

Get the order right and agents stop needing hand-holding.
Get it wrong and you're just building the wrong thing faster.
